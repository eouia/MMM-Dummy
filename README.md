# MMM-Dummy
dummy module for just occupation of position

## Installation
```
cd ~/MagicMirror/modules
git clone https://github.com/eouia/MMM-Dummy
```

## Configuration

```

{
  module:"MMM-Dummy",
  position: "fullscreen_below",
  config: {
    width: "100vw",
    height: "100vh",
  }
}
```


## What this module can do?
Nothing. It just exists.
